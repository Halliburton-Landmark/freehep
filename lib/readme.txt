Libraries used when running FREEHEP

This directory contains all libraries needed when you RUN freehep. Libraries only
used for compilation should go in the tools directory.


Each library specfies name, version number, description, their origin and their use(rs)

(alphabetical)
freehep-base.jar
    2.0-SNAPSHOT
    from our maven 2 repository
    missing parts are preview

aida.jar, aida-dev.jar, aida-test.jar
    3.2.1
    Abstract Interfaces for Data Analysis
    http://aida.freehep.org/
    jaida, ...

bcel.jar
    5.1-dev (built from CVS source by tonyj, 4/23/2003)
    Byte-Code Engineering Library
    http://jakarta.apache.org/bcel/index.html
    rootio, jace

commons-collections.jar
    2.1
    Collection classes from Apache
    http://jakarta.apache.org/commons/
    velocity.jar

cprep.jar
    no longer supported
    IBM Alphaworks
    jaco

jdom.jar
    1.0
    A Java representation of an XML document
    http://www.jdom.org
    Application framework, xmlio, ...

jel.jar
    0.9.10
    Java Expressions Library
    http://galaxy.fzu.cz/JEL/
    fitting

jlfgr.jar
    1.0
    The Java Look and Feel Graphics
    http://developer.java.sun.com/developer/techDocs/hi/repository/
    icon browser

jh.jar
    1.1.3
    Java Help System
    http://www.javasoft.com/products/javahelp
    ???

l2fprod-common-all
    0.1
    Provides extra Swing components and a component to edit Properties in a table-like user interface.
    http://common.L2FProd.com
    hep.aida.ref.plotter.styleEditor.AidaStyleEditor

logkit.jar
    1.2.2
    Logging classes from Avalon Apache
    http://avalon.apache.org/logkit/
    velocity.jar

optimizer.jar
    ?
    A few optimizers.
    http://www1.fpl.fs.fed.us/optimization.html
    jaida

openide-lookup.jar
    1.9 + patches
    The NetBeans lookup library
    http://openide.netbeans.org/lookup/index.html
    application framework

servlet.zip:
    2.3
    Java Servlet classes, to be able to compile the servlets
    http://www.javasoft.com/products/servlet/
    yappi servlet

velocity.jar
    1.3.1
    Velocity Template Engine
    http://jakarta.apache.org/velocity
    webpage
