Browse Particle Data

The Java Servlet provides online information about the particle properties. Quick particle searches can be performed by specifying particle name or PDG ID number.

The YaPPI Servlet will undergo a number of improvements (see To Do List), but at the moment, you may browse particle data by name, e.g by typing "eta(c)(1S)", "eta", "pi+", and so on, in the name box below.
